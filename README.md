# Juego de Anatomía Humana en 3D

## Descripción

Videojuego educativo desarrollado en Unity con el objetivo de facilitar el aprendizaje de la anatomía humana mediante modelos tridimensionales e interacción visual.

## Características

* Visualización de estructuras anatómicas en 3D.
* Interacción con modelos del cuerpo humano.
* Experiencia educativa e interactiva.
* Navegación por diferentes sistemas anatómicos.
* Diseño orientado al aprendizaje.

## Tecnologías Utilizadas

* Unity
* C#
* HTML/CSS (para recursos o contenido complementario, si aplica)

## Objetivos del Proyecto

* Aplicar conceptos de desarrollo de videojuegos.
* Utilizar gráficos tridimensionales con fines educativos.
* Mejorar la experiencia de aprendizaje mediante la interacción visual.

## Futuras Mejoras

* Sistema de evaluación del aprendizaje.
* Más modelos anatómicos.
* Compatibilidad con dispositivos móviles.
* Mayor nivel de interacción con los órganos y sistemas.
